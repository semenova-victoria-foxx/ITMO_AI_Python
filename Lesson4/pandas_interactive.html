Библиотека pandas

Pandas — библиотека Python для обработки и анализа табличных данных. По духу она напоминает Excel или Google Sheets, но даёт больше гибкости и позволяет работать кодом.

  • основные структуры: Series и DataFrame;
  • данные можно читать из CSV и других файлов;
  • типичные задачи: просмотр, фильтрация, выбор строк и столбцов, сортировка.
import pandas as pd

Series

Series — это одномерная структура данных с индексами. Слева индекс, справа значение.

a = pd.Series([4, 7, 6, 3, 9],
              index=['one', 'two', 'three', 'four', 'five'])
Можно думать о Series как об одном столбце таблицы.

DataFrame

DataFrame — табличная структура: строки + столбцы. В разных столбцах могут быть разные типы данных.

df = pd.DataFrame({
    'Age': [46, 37, 44, 42, 42],
    'Country': ['Spain', 'Spain', 'Germany', 'Germany', 'France'],
    'Gender': ['Female', 'Female', 'Male', 'Male', 'Male']
})
Отдельный столбец DataFrame — это Series.

Отображение данных

  • pd.read_csv(...) — чтение CSV;
  • df.head() — первые строки таблицы;
  • df.shape — размер таблицы: (строки, столбцы).
df = pd.read_csv('./Churn_Modelling.csv')
df.head()
df.shape

Первичный анализ данных

  • df.info() — общая информация о таблице и типах столбцов;
  • df.describe() — статистика по числовым столбцам;
  • df.describe(include='object') — описание категориальных столбцов;
  • value_counts() — частоты значений в столбце.

Фильтрация

Фильтрация в pandas строится на булевых масках.

df['Gender'] == 'Male'
df[df['Gender'] == 'Male']
  • & — логическое И;
  • | — логическое ИЛИ;
  • ~ — логическое НЕ;
  • isin(...) — проверка вхождения в список значений.
Сложные условия в pandas удобно писать в круглых скобках.

Индексация: loc и iloc

  • loc берёт данные по именам строк и столбцов;
  • iloc берёт данные по позициям строк и столбцов.
df_small.loc[1]
df_small.loc[[1, 4, 5], ['Gender', 'Age']]

df_small.iloc[[0, 1, 2]]
df_small.iloc[0, [0, 2]]

Сортировки

  • sort_values('Age') — по возрастанию;
  • sort_values('Age', ascending=False) — по убыванию;
  • можно сортировать по нескольким столбцам сразу.
df.sort_values('Age')
df.sort_values('Age', ascending=False)
df.sort_values(['Age', 'Tenure'], ascending=[True, False])
